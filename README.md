# Control-new
Water cooling power control  
基于温度传感器的水冷控制  
PS：此版用于水冷控制，上一版用于[风冷](https://github.com/Yirami/Control-old)

---
## 说明
采集DS18b20传感器的温度数据，并依据设定的阈值，自动调节水泵功率。
+ `./3D/`下为基于ProE/CREO建立的水冷箱三维模型  
+ `./pcb/`下为PCB工程*（基于Altium Designer）*  
+ `./src/`下为源代码*（C语言）*
